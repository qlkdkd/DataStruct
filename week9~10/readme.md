# 리스트 추상 데이터 타입
## 리스트의 소개
* 리스트: 우리들이 자료를 정리하는 방법 중의 하나
* 일상생활에서의 리스트
![image](https://github.com/qlkdkd/DataStructure/assets/71871927/f4a48a5c-643a-41c2-8ac3-850b89731f68)

## 리스트의 기본 연산
$$L=(item_0, item_1, item_2,...,item_{n-1}$$

### 연산 목록
* 리스트에 새로운 항목을 추가(삽입 연산)
* 리스트에 항목을 삭제(삭제 연산)
* 리스트에서 특정한 항목 찾기(탐색 연산)

## 리스트의 ADT
